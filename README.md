# MatchFind

**MatchFind** is a premium, modern matchmaking application designed to help users find their perfect life partner (Rishtey) with ease and elegance. It features a robust profile management system, advanced search capabilities, and a seamless user experience across both web and mobile platforms.

### **Authentication & Security**

- **Secure Signup/Login**: Powered by Supabase with email confirmation logic.
- **Enhanced UX**: Playful "lift" animations on inputs, auto-formatting for phone numbers (+91), and real-time password validation with visual feedback.
- **Mobile Optimized**: "Get App" modal for Android users and smooth mobile-first animations.

### **Dashboard & Discovery**

- **Smart Navigation**: Floating navigation bar (Profile ‚Üí Rishtey ‚Üí Search ‚Üí Matches) for effortless browsing.
- **Rishtey Discovery**: Interactive "Finding Rishtey..." spinner animation while fetching potential matches.
- **Advanced Matching**: View 'You Liked', 'Liked You', 'Matches', and 'Rejected' profiles in dedicated tabs.

### **Profile Management**

- **Comprehensive Profiles**: detailed sections for Basic Info, Education, Career, Family, and Location.
- **Dynamic Theming**: Gender-based UI themes (Blue/Cyan for Men, Rose/Orange for Women).
- **Smart Editing**: Partial field editing with direct navigation and "square-pen" shortcuts.
- **Visuals**: Circular profile pictures with image pop-out viewer for high-resolution inspection.

### **Form & Input Intelligence**

- **Auto-Formatting**: Height (x.y ‚Üí x.0y), Title Case names, and auto-prefixed phone numbers.
- **Smart Components**: Keyboard-navigable Autocomplete for States, AM/PM TimePicker, and occupation-specific fields (Job vs Business).
